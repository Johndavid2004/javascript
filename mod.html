Document.getElementById()

Este método es el más sencillo y eficiente para seleccionar un solo elemento HTML en un documento. Se utiliza para obtener un elemento específico basado en su atributo id, que debe ser único dentro de la página.

Busca en el documento HTML el elemento que coincida con el valor del id proporcionado. Retorna solo un elemento, ya que el id es único en el documento. Si no encuentra ningún elemento con el id especificado, devuelve null. Ventajas: Es extremadamente rápido porque el id es único y no requiere recorrer todo el DOM. Ideal cuando necesitas manipular un solo elemento que tiene un id predefinido.

Document.getElementsByClassName()

Este método devuelve una colección (similar a un array) de todos los elementos que tienen una clase específica. Es útil cuando necesitas seleccionar varios elementos que comparten la misma clase.

Busca todos los elementos en el documento que contengan la clase proporcionada. Devuelve una colección de nodos (HTMLCollection), que es dinámica, lo que significa que si el DOM cambia (por ejemplo, si se añade o elimina un elemento con esa clase), la colección se actualizará automáticamente. La colección es iterable, pero no es un array verdadero. Puedes acceder a los elementos usando índices (elementos[0], elementos[1], etc.) y recorrerla con bucles como for o for...of. Si no encuentra ningún elemento, devuelve una colección vacía.

Document.getElementsByTagName()

Este método se utiliza para seleccionar todos los elementos de un determinado tipo de etiqueta, como div, p, a, etc.

Selecciona todos los elementos que coinciden con el nombre de la etiqueta especificada. Devuelve una colección de nodos (HTMLCollection) de todos los elementos que coinciden con el nombre de la etiqueta. Es útil para seleccionar todos los elementos de un mismo tipo, como todas las imágenes.

Document.querySelector()

Este método es más versátil y flexible que los anteriores, ya que puede seleccionar el primer elemento que coincida con un selector CSS. Te permite seleccionar un elemento usando cualquier selector que sea válido en CSS, como #id, .clase, etiqueta, combinaciones de ellos, pseudoclases, etc.

Busca el primer elemento en el documento que coincida con el selector CSS especificado. Acepta cualquier selector válido de CSS, como #miId, .miClase, div > p, etc. Si encuentra el elemento, lo devuelve. Si no lo encuentra, devuelve null. Es más versátil que getElementById, getElementsByClassName, o getElementsByTagName, ya que puedes usar selectores CSS complejos.

Document.querySelectorAll()

Este método es similar a querySelector, pero en lugar de devolver solo el primer elemento que coincide con el selector, devuelve todos los elementos que coinciden con el selector CSS especificado.

Selecciona todos los elementos que coincidan con el selector CSS proporcionado y devuelve una NodeList. A diferencia de getElementsByClassName o getElementsByTagName, que devuelven colecciones vivas, una NodeList es estática, lo que significa que no se actualizará automáticamente si el DOM cambia. Puedes usar forEach() para recorrer una NodeList, lo que facilita la manipulación de múltiples elementos a la vez. Es extremadamente útil cuando se necesitan seleccionar múltiples elementos utilizando selectores CSS complejos o avanzados. Si no encuentra ningún elemento, devuelve una NodeList vacía.
